========================================
  美国天气预报 - 安装指南（3步搞定）
========================================

功能：
- 每天早上7:30自动发送美国8大区域16个城市的7天天气预报到你的邮箱
- 每6小时检测一次极端天气（龙卷风、洪水、暴风雪等），发现就发邮件提醒
- 完全免费，不用服务器，不用写代码


========================================
第1步：复制仓库到你的GitHub
========================================

1. 打开浏览器，访问：
   https://github.com/lic94072-hash/us-weather-push

2. 点右上角「Fork」按钮

3. 弹出页面直接点绿色按钮「Create fork」

4. 等几秒钟，代码就复制到你的账号下了


========================================
第2步：开启QQ邮箱授权码
========================================

1. 浏览器打开 https://mail.qq.com 并登录你的QQ邮箱

2. 点顶部的「设置」→「账户」

3. 往下翻页面，找到这一行：
   「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务」

4. 找到「IMAP/SMTP服务」，点右边的「开启」

5. 按提示用手机发一条短信验证

6. 验证成功后，页面会显示一个16位的授权码
   格式类似：abcdefghijklmnop

   *** 这个授权码只显示一次！一定要复制保存好！***

   如果忘了，只能关掉重新开，会生成新的授权码。


========================================
第3步：填写5个密钥（Secrets）
========================================

1. 打开你刚Fork的仓库页面
   格式：https://github.com/你的用户名/us-weather-push

2. 点上方「Settings」（设置）

3. 左侧菜单点「Secrets and variables」→「Actions」

4. 点右上角「New repository secret」

5. 逐个添加以下5个，每添加一个点一次「Add secret」：

   第1个：
     Name 填：EMAIL_SMTP
     Value 填：smtp.qq.com
     → 点 Add secret

   第2个：
     Name 填：EMAIL_PORT
     Value 填：465
     → 点 Add secret

   第3个：
     Name 填：EMAIL_SENDER
     Value 填：你的QQ邮箱（比如 123456789@qq.com）
     → 点 Add secret

   第4个：
     Name 填：EMAIL_PASSWORD
     Value 填：第2步获取的16位授权码
     → 点 Add secret

   第5个：
     Name 填：EMAIL_RECEIVER
     Value 填：接收邮件的邮箱（可以和发件邮箱一样）
     → 点 Add secret

   注意：
   - Name 不能用中文、不能有空格、不能用减号-
   - 只能用英文字母和下划线_
   - Value 里的授权码不是QQ密码，是第2步获取的16位码


========================================
测试一下
========================================

1. 打开你的仓库页面

2. 点上方「Actions」

3. 左边点「US Weather Push」

4. 右边点「Run workflow」→ 弹框直接点绿色「Run workflow」

5. 等待1~2分钟，刷新页面看结果

6. 如果出现绿色勾✓，去QQ邮箱看看，应该收到天气预报邮件了


========================================
收到邮件就成功了！
========================================

以后不用管了，自动运行：

- 每天：
  北京时间 7:30 自动发7天天气预报邮件

- 每6小时：
  自动检测极端天气，只有发现严重/极端预警才发邮件提醒
  如果天气正常，不会打扰你

如果你想手动触发：
  仓库 → Actions → 选对应的工作流 → Run workflow

========================================
常见问题
========================================

Q：收不到邮件？
A：检查QQ邮箱授权码是否正确，检查Secrets里5个值是否都填了

Q：Actions页面没有绿色勾？
A：点进失败的运行记录看日志，一般是授权码填错了

Q：能不能发给多个邮箱？
A：可以，收件箱设置转发规则即可

Q：费用？
A：全部免费。天气API免费，GitHub Actions免费，QQ邮箱免费

Q：想改推送时间？
A：编辑仓库里 .github/workflows/weather-push.yml，改cron那一行

Q：想加城市？
A：编辑仓库里 config.py，按格式添加城市名和经纬度
